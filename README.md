## English

### 🌟 What is overleaf-cli?

Browser plugin + backend that enables AI tools like **Claude Code**, **Cursor**, **Windsurf**, and **GitHub Copilot** to directly edit Overleaf projects through local file synchronization.

### ✨ Key Features

- **Zero Extra Cost** - No subscriptions needed, no token costs
- **Stay in Your Flow** - Keep using Overleaf and your AI tools. We just connect them seamlessly
- **Lightning Fast** - Direct DOM manipulation, no browser automation overhead
- **High Accuracy** - 98.5% task success rate, 3.8x faster than browser automation tools

### 🚀 Installation

#### Backend (CLI Tool)

```bash
npm install -g overleaf-cli
```

Start the server:
```bash
overleaf-cli start
```

#### Frontend (Browser Extension)

1. Download the extension from [GitHub Releases](https://github.com/BruceChenSF/overleaf-cli/releases)
2. Load it in Chrome/Edge:
   - Go to \`chrome://extensions/\`
   - Enable Developer Mode
   - Click "Load unpacked"
   - Select the extension directory

### 📖 How It Works

1. **Browser Plugin** - Captures Overleaf edits and syncs to local folder
2. **Local Sync** - Files stored in local project directory
3. **AI Operations** - Your AI tool edits files locally
4. **DOM Update** - Backend detects changes and updates Overleaf directly

- Cursor

- And more...

## 中文

### 🌟 什么是 overleaf-cli？

浏览器插件 + 后端，让 **Claude Code**、**Cursor**、**Windsurf** 和 **GitHub Copilot** 等AI工具可以通过本地文件同步直接编辑 Overleaf 项目。

### ✨ 核心特性

- **零额外成本** - 无需订阅，无需token费用
- **保持工作流** - 继续使用Overleaf和您的AI工具，我们只是无缝连接它们
- **闪电般快速** - 直接DOM操作，无浏览器自动化开销
- **高准确率** - 98.5%任务成功率，比浏览器自动化工具快3.8倍

### 🚀 安装

#### 后端（CLI工具）

```bash
npm install -g overleaf-cli
```

启动服务器：
```bash
overleaf-cli start
```

#### 前端（浏览器扩展）

1. 从 [GitHub Releases](https://github.com/BruceChenSF/overleaf-cli/releases) 下载扩展
2. 在 Chrome/Edge 中加载：
   - 打开 \`chrome://extensions/\`
   - 启用开发者模式
   - 点击"加载已解压的扩展程序"
   - 选择扩展目录

### 📖 工作原理

1. **浏览器插件** - 捕获Overleaf编辑并同步到本地文件夹
2. **本地同步** - 文件存储在本地项目目录
3. **AI操作** - AI工具在本地编辑文件
4. **DOM更新** - 后端检测更改并直接更新Overleaf

- 更多...
